# Битови полета

## 1. Основна идея

Позволяват дефиниране на членове с конкретен брой битове.

```cpp
struct Flags {
    unsigned int a : 1;
    unsigned int b : 1;
    unsigned int c : 1;
};
```

Тук `a`, `b` и `c` са по 1 бит.

## 2. Разположение

Компилаторът групира битовите полета в една машинна дума.

Пример:

```cpp
struct Flags {
    unsigned int a : 1;
    unsigned int b : 3;
    unsigned int c : 4;
};
```

Общо: 8 бита.

Възможно разположение:

```
| cccc | bbb | a |
```

Но:

- Подредбата на битовете е implementation-defined
- Не е гарантирано между различни компилатори

## 3. Ограничения

- Не може да вземем адрес на битово поле
- Не може да бъде `static`
- Не може да бъде референция

```cpp
// Грешка
unsigned int* p = &flags.a;
```

## 4. Практическа употреба

- Хардуерни регистри
- Компактни флагове
- Мрежови протоколи (с внимание към *endian*-ността)
